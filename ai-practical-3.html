What is A* Search?

A* (A-star) is an informed search algorithm that finds the optimal path from start to goal by using both:

  • g(n): Actual cost from start to current node
  • h(n): Heuristic estimate from current node to goal
f(n) = g(n) + h(n)

Key Components:

Cost Function g(n): The actual distance traveled from Bharuch to the current city via roads.
Heuristic Function h(n): Straight-line distance from current city to Saputara (admissible heuristic).
Evaluation Function f(n): Total estimated cost of the cheapest path through current node.

Algorithm Properties:

  • Optimal: Guaranteed to find shortest path (if heuristic is admissible)
  • Complete: Will find solution if one exists
  • Efficient: Uses heuristic to guide search toward goal
  • Informed: Makes better decisions than uninformed search
Real-world Application: A* is used in GPS navigation systems, video game AI pathfinding, robotics, and network routing protocols to find optimal paths efficiently.

# A* Search using heapq
import heapq

def a_star_search(graph, start, goal, heuristic, costs):
    """A* Search: Best-First + actual costs"""
    heap = [(0 + heuristic[start], 0, [start])]
    visited = set()
    
    while heap:
        f_cost, g_cost, path = heapq.heappop(heap)
        current = path[-1]
        
        if current == goal:
            return path, g_cost
            
        if current not in visited:
            visited.add(current)
            
            for neighbor in graph[current]:
                if neighbor not in visited:
                    new_g = g_cost + costs.get((current, neighbor), 1)
                    new_f = new_g + heuristic[neighbor]
                    new_path = path + [neighbor]
                    
                    heapq.heappush(heap, (new_f, new_g, new_path))
    
    return None, float('inf')
