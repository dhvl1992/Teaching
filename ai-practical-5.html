📚 Theory: Game Tree Search

What is Minimax?

Minimax is a decision-making algorithm for turn-based games that assumes both players play optimally:

  • Maximizing Player: Tries to get the highest score (AI)
  • Minimizing Player: Tries to get the lowest score (Human)
  • Game Tree: Explores all possible future game states
Minimax(node) = max(children) if maximizing
Minimax(node) = min(children) if minimizing

Alpha-Beta Pruning:

Alpha (α): Best value maximizer can guarantee (lower bound)
Beta (β): Best value minimizer can guarantee (upper bound)
Pruning: If α ≥ β, stop exploring (cut-off occurs)

Algorithm Properties:

  • Optimal: Finds the best possible move
  • Complete: Always terminates for finite games
  • Efficient: Alpha-Beta reduces search space by ~50%
  • Perfect Play: AI never loses if it goes first
Scoring System: Win = +10, Lose = -10, Draw = 0. The AI maximizes score while assuming human minimizes it.

💻 Minimax Implementation

# Simple Minimax - Minimal Code (No Functions)

# Board: 9 cells, '' = empty, 'X' = human, 'O' = AI
b = [''] * 9

# Check winner - all winning combinations
wins = [[0,1,2],[3,4,5],[6,7,8],[0,3,6],[1,4,7],[2,5,8],[0,4,8],[2,4,6]]
winner = None
for w in wins:
    if b[w[0]] == b[w[1]] == b[w[2]] != '':
        winner = b[w[0]]

# Minimax recursive logic (inline)
scores = []
for i in range(9):
    if b[i] == '':  # Try AI move
        b[i] = 'O'
        
        # Check if AI wins
        ai_wins = False
        for w in wins:
            if b[w[0]] == b[w[1]] == b[w[2]] == 'O':
                ai_wins = True
        if ai_wins:
            scores.append(10)
            b[i] = ''
            continue
            
        # Try all human responses
        min_score = 999
        for j in range(9):
            if b[j] == '':  # Human move
                b[j] = 'X'
                
                # Check if human wins
                human_wins = False
                for w in wins:
                    if b[w[0]] == b[w[1]] == b[w[2]] == 'X':
                        human_wins = True
                if human_wins:
                    min_score = -10
                else:
                    min_score = min(min_score, 0)  # Draw
                
                b[j] = ''  # Undo
        
        scores.append(min_score)
        b[i] = ''  # Undo

# Find best move
best_move = scores.index(max(scores))
print(f"AI plays: {best_move}")

# Complete Minimax Game - Ultra Simple

# Setup
b = [''] * 9  # Empty board
wins = [[0,1,2],[3,4,5],[6,7,8],[0,3,6],[1,4,7],[2,5,8],[0,4,8],[2,4,6]]

# Game loop
while True:
    # Show board
    for i in range(0,9,3):
        print(f" {b[i] if b[i] else i} | {b[i+1] if b[i+1] else i+1} | {b[i+2] if b[i+2] else i+2} ")
        if i < 6: print("-----------")
    
    # Human move
    move = int(input("Your move (0-8): "))
    if b[move] == '': b[move] = 'X'
    
    # Check win/draw
    game_over = False
    for w in wins:
        if b[w[0]] == b[w[1]] == b[w[2]] != '':
            print(f"{b[w[0]]} wins!"); game_over = True
    if '' not in b: print("Draw!"); game_over = True
    if game_over: break
    
    # AI move - Simple Minimax
    best = -999; best_move = 0
    for i in range(9):
        if b[i] == '':
            b[i] = 'O'  # Try AI move
            
            # AI win check
            score = 0
            for w in wins:
                if b[w[0]] == b[w[1]] == b[w[2]] == 'O': score = 10
            
            # If not AI win, check human responses
            if score == 0:
                worst = 999
                for j in range(9):
                    if b[j] == '':
                        b[j] = 'X'
                        human_score = 0
                        for w in wins:
                            if b[w[0]] == b[w[1]] == b[w[2]] == 'X': human_score = -10
                        worst = min(worst, human_score)
                        b[j] = ''
                score = worst if worst != 999 else 0
            
            if score > best: best = score; best_move = i
            b[i] = ''  # Undo
    
    b[best_move] = 'O'
    print(f"AI plays: {best_move}")
    
    # Check win again
    for w in wins:
        if b[w[0]] == b[w[1]] == b[w[2]] != '':
            print(f"{b[w[0]]} wins!"); game_over = True
    if '' not in b: print("Draw!"); game_over = True
    if game_over: break
