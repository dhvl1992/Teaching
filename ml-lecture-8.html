Why ANNs Struggle with Images

‚ùå No Spatial Structure: Treats images as flat vectors
‚ùå Too Many Parameters: 784√ó64 = 50,240 weights in first layer
‚ùå Position Dependent: Moving digit changes all connections
‚ùå No Translation Invariance: Same digit in different positions = different patterns
‚ùå Overfitting Prone: High parameter count leads to memorization

Why CNNs Excel at Images

‚úÖ Parameter Sharing: Same 3√ó3 filter used everywhere
‚úÖ Spatial Awareness: Preserves 2D structure
‚úÖ Translation Invariant: Detects edges/shapes anywhere
‚úÖ Hierarchical Learning: Edges ‚Üí Shapes ‚Üí Objects
‚úÖ Efficient: 82% fewer parameters for same task
