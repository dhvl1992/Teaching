üìö Theory: Rule-Based Systems

What is a Rule-Based System?

A rule-based system (or production system) is a type of artificial intelligence system that uses a set of if-then rules to represent knowledge and make decisions. It's one of the earliest forms of AI and is still widely used today.

Key Components:

  • Knowledge Base: Collection of rules (IF-THEN statements)
  • Working Memory: Current facts and data
  • Inference Engine: Applies rules to derive new facts
  • User Interface: Interaction with the system

Forward Chaining:

A reasoning method that starts with known facts and applies rules to derive new facts until a goal is reached. It's data-driven and works from facts to conclusions.

Example Rules for Light Diagnosis:

  • IF light_switch_on = False THEN cause = "Switch is off"
  • IF light_switch_on = True AND bulb_working = False THEN cause = "Bulb is fused"
  • IF light_switch_on = True AND bulb_working = True AND power_available = False THEN cause = "No power supply"

Rule-Based System Implementation

# Rule-Based System for Light Diagnosis

print("=== Light Diagnosis Expert System ===")
print("Answer the following questions to diagnose the light problem:")

# Step 1: Create a dictionary to store facts (what we know)
facts = {}

# Step 2: Get user input and store facts
switch_answer = input("Is the light switch turned on? (yes/no): ")
if switch_answer.lower() == 'yes':
    facts['light_switch_on'] = True
else:
    facts['light_switch_on'] = False

# Step 3: Apply Rule 1 - Check if switch is off
if facts['light_switch_on'] == False:
    diagnosis = "Switch is off - Please turn on the light switch"
    print(f"\nüîç Diagnosis: {diagnosis}")
    print(f"Rule applied: IF light_switch_on = False THEN cause = Switch is off")
    print(f"Facts used: {facts}")
else:
    # Continue asking questions if switch is on
    bulb_answer = input("Is the bulb working (not burnt out)? (yes/no): ")
    if bulb_answer.lower() == 'yes':
        facts['bulb_working'] = True
    else:
        facts['bulb_working'] = False
    
    # Step 4: Apply Rule 2 - Check if bulb is fused
    if facts['light_switch_on'] == True and facts['bulb_working'] == False:
        diagnosis = "Bulb is fused - Please replace the light bulb"
        print(f"\nüîç Diagnosis: {diagnosis}")
        print(f"Rule applied: IF light_switch_on = True AND bulb_working = False THEN cause = Bulb is fused")
        print(f"Facts used: {facts}")
    else:
        # Continue asking if bulb is working
        power_answer = input("Is there power available (other lights working)? (yes/no): ")
        if power_answer.lower() == 'yes':
            facts['power_available'] = True
        else:
            facts['power_available'] = False
        
        # Step 5: Apply Rule 3 - Check power supply
        if facts['light_switch_on'] == True and facts['bulb_working'] == True and facts['power_available'] == False:
            diagnosis = "No power supply - Check circuit breaker or electrical connection"
            print(f"\nüîç Diagnosis: {diagnosis}")
            print(f"Rule applied: IF light_switch_on = True AND bulb_working = True AND power_available = False THEN cause = No power supply")
            print(f"Facts used: {facts}")
        # Step 6: Apply Rule 4 - Everything else working
        elif facts['light_switch_on'] == True and facts['bulb_working'] == True and facts['power_available'] == True:
            diagnosis = "Light fixture problem - Check wiring or consult an electrician"
            print(f"\nüîç Diagnosis: {diagnosis}")
            print(f"Rule applied: IF light_switch_on = True AND bulb_working = True AND power_available = True THEN cause = Light fixture problem")
            print(f"Facts used: {facts}")
        else:
            print(f"\nüîç No diagnosis found")
            print(f"Facts used: {facts}")

print("\n=== Diagnosis Complete ===")
print("This is how a rule-based system works!")
print("It uses IF-THEN rules to make decisions based on facts.")
