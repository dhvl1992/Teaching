📚 Theory: Uninformed Search

What is State-Space Search?

State-space search is a problem-solving method where we define a problem in terms of states (possible configurations) and actions (transitions between states). The goal is to find a sequence of actions that leads from an initial state to a goal state.

Breadth-First Search (BFS):

  • Strategy: Explores all nodes at depth d before exploring nodes at depth d+1
  • Data Structure: Uses a queue (FIFO - First In, First Out)
  • Optimality: Guarantees the shortest path
  • Completeness: Will find a solution if one exists

Depth-First Search (DFS):

  • Strategy: Explores as far as possible along each branch before backtracking
  • Data Structure: Uses a stack (LIFO - Last In, First Out) or recursion
  • Optimality: Does not guarantee the shortest path
  • Completeness: May get stuck in infinite loops without cycle detection

Key Differences:

Aspect BFS DFS
Path Quality Optimal (shortest) May not be optimal
Memory Usage Higher Lower
Search Pattern Level by level Deep exploration

BFS Implementation (Level by Level)

# BFS - Breadth First Search (Level by Level)

# Step 1: Set up the maze
maze = [
    ['S', 0, 1, 0],
    [0, 0, 1, 0],
    [1, 0, 0, 0],
    [0, 1, 0, 'E']
]

# Step 2: Find start and end positions
start_row, start_col = 0, 0  # Position of 'S'
end_row, end_col = 3, 3      # Position of 'E'

# Step 3: Set up BFS
queue = []                    # Store positions to visit
queue.append((start_row, start_col))
visited = []                  # Remember visited positions

# Step 4: BFS main loop
print("=== BFS Search Process ===")
while queue:
    # Take the first position from queue (FIFO)
    current_row, current_col = queue.pop(0)
    
    # Mark as visited
    visited.append((current_row, current_col))
    print(f"Visiting: ({current_row}, {current_col})")
    
    # Check if we found the end
    if current_row == end_row and current_col == end_col:
        print("Found the end! BFS Complete!")
        break
    
    # Look at all 4 directions (up, down, left, right)
    directions = [(-1, 0), (1, 0), (0, -1), (0, 1)]
    
    for dr, dc in directions:
        new_row = current_row + dr
        new_col = current_col + dc
        
        # Check if new position is valid
        if (0 <= new_row < 4 and 0 <= new_col < 4 and           # Inside maze
            maze[new_row][new_col] != 1 and                     # Not a wall
            (new_row, new_col) not in visited and               # Not visited
            (new_row, new_col) not in queue):                   # Not in queue
            
            queue.append((new_row, new_col))
            print(f"Added to queue: ({new_row}, {new_col})")

print(f"Total positions visited: {len(visited)}")
print(f"Visited order: {visited}")

DFS Implementation (Deep First)

# DFS - Depth First Search (Deep First)

# Step 1: Set up the maze (same as BFS)
maze = [
    ['S', 0, 1, 0],
    [0, 0, 1, 0],
    [1, 0, 0, 0],
    [0, 1, 0, 'E']
]

# Step 2: Find start and end positions
start_row, start_col = 0, 0  # Position of 'S'
end_row, end_col = 3, 3      # Position of 'E'

# Step 3: Set up DFS
stack = []                    # Store positions to visit
stack.append((start_row, start_col))
visited = []                  # Remember visited positions

# Step 4: DFS main loop
print("=== DFS Search Process ===")
while stack:
    # Take the last position from stack (LIFO)
    current_row, current_col = stack.pop()
    
    # Skip if already visited
    if (current_row, current_col) in visited:
        continue
    
    # Mark as visited
    visited.append((current_row, current_col))
    print(f"Visiting: ({current_row}, {current_col})")
    
    # Check if we found the end
    if current_row == end_row and current_col == end_col:
        print("Found the end! DFS Complete!")
        break
    
    # Look at all 4 directions (up, down, left, right)
    directions = [(-1, 0), (1, 0), (0, -1), (0, 1)]
    
    for dr, dc in directions:
        new_row = current_row + dr
        new_col = current_col + dc
        
        # Check if new position is valid
        if (0 <= new_row < 4 and 0 <= new_col < 4 and           # Inside maze
            maze[new_row][new_col] != 1 and                     # Not a wall
            (new_row, new_col) not in visited):                 # Not visited
            
            stack.append((new_row, new_col))
            print(f"Added to stack: ({new_row}, {new_col})")

print(f"Total positions visited: {len(visited)}")
print(f"Visited order: {visited}")

Simple Comparison

# Key Differences Between BFS and DFS

print("=== BFS vs DFS Comparison ===")

# BFS uses Queue (First In, First Out)
bfs_queue = [1, 2, 3]
first_out = bfs_queue.pop(0)  # Takes 1 (first added)
print(f"BFS Queue: {bfs_queue}, First out: {first_out}")

# DFS uses Stack (Last In, First Out)  
dfs_stack = [1, 2, 3]
last_out = dfs_stack.pop()    # Takes 3 (last added)
print(f"DFS Stack: {dfs_stack}, Last out: {last_out}")

print("\nBFS Characteristics:")
print("- Explores level by level (like ripples in water)")
print("- Uses queue: First In, First Out")
print("- Finds shortest path")
print("- Uses more memory")

print("\nDFS Characteristics:")
print("- Explores deep first (like going down a tunnel)")
print("- Uses stack: Last In, First Out")
print("- May not find shortest path")
print("- Uses less memory")

print("\nWhen to use:")
print("- BFS: When you need the shortest path")
print("- DFS: When you need to explore all possibilities")
