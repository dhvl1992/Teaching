Uninformed Search

Folder Search with BFS and DFS

🎯 Aim

To implement and compare uninformed search algorithms (BFS and DFS) for finding files in a computer's folder system.

Real-Life Scenario: Finding a File on Your Computer

  • Think of searching for a file in your computer's folders and subfolders
  • Each folder is like a "node" and contains files or more subfolders
  • BFS explores all folders at one level before diving deeper (like checking Desktop, Downloads, Documents first)
  • DFS goes deep into one folder path before trying others (like diving into Documents → College → Projects → Final)
  • Everyone can relate to this - we all search for files daily!

📚 Theory: Search Algorithms in File Systems

What is File System Search?

When you search for a file on your computer, the system needs to look through folders systematically. This is exactly what BFS and DFS algorithms do - they provide different strategies for exploring the folder tree structure.

Breadth-First Search (BFS) - Level by Level:

  • Like checking all folders on your desktop first, then diving into each one
  • Uses a queue: "Remember to check these folders later"
  • Finds files faster if they're in shallow folders (not buried deep)
  • Systematic approach: Guarantees you'll find the file in minimum folder depth

Depth-First Search (DFS) - Deep Dive:

  • Like opening one folder and going as deep as possible before trying another
  • Uses a stack: "Let me finish exploring this path first"
  • Memory efficient: Doesn't need to remember many folders at once
  • May find files quickly if you pick the right path first

Real-World Comparison:

Scenario BFS (Level by Level) DFS (Deep Dive)
Looking for recent downloads Perfect - checks Downloads folder first Might waste time in deep subfolders
Finding old project files Takes time to reach deep folders Great if you remember the path
Memory usage Remembers many folders Only remembers current path

🔍 BFS - Breadth First Search (Check All Folders at Each Level)

# BFS: Like checking all folders on desktop first, then diving deeper

def search_file_with_bfs(file_system, target_file):
    print("=== 🔍 BFS: Level by Level Search ===")
    print("Strategy: Check all folders at current level before going deeper")
    print()
    
    # Queue: "Remember to check these folders later"
    folders_to_check = []  # This is our queue
    folders_to_check.append(("Root", file_system["Root"], ["Root"]))
    
    steps_taken = 0
    
    while folders_to_check:
        # Take the first folder from our queue (FIFO - First In, First Out)
        current_name, current_folder, path = folders_to_check.pop(0)
        steps_taken += 1
        
        print(f"Step {steps_taken}: 📂 Checking folder '{current_name}'")
        print(f"Path: {' → '.join(path)}")
        
        # Look through everything in this folder
        if "contains" in current_folder:
            for item_name, item_data in current_folder["contains"].items():
                
                # Found a file!
                if item_data["type"] == "file":
                    print(f"   📄 Found file: {item_name}")
                    
                    # Is this the file we want?
                    if item_name == target_file:
                        final_path = path + [item_name]
                        print(f"   ✅ SUCCESS! Found '{target_file}'!")
                        print(f"   📍 Location: {' → '.join(final_path)}")
                        print(f"   📊 Total folders checked: {steps_taken}")
                        print(f"   🎯 BFS guarantees shortest folder path!")
                        return final_path
                
                # Found another folder - add it to queue for later
                elif item_data["type"] == "folder":
                    new_path = path + [item_name]
                    folders_to_check.append((item_name, item_data, new_path))
                    print(f"   📂 Added '{item_name}' to queue for later checking")
        
        print()  # Empty line for readability
    
    print(f"❌ File '{target_file}' not found anywhere!")
    print(f"📊 Total folders checked: {steps_taken}")
    return None

# Run the search!
result = search_file_with_bfs(file_system, "vacation.jpg")

🏃‍♂️ DFS - Depth First Search (Deep Dive Strategy)

# DFS: Like opening one folder and going as deep as possible

def search_file_with_dfs(file_system, target_file):
    print("=== 🏃‍♂️ DFS: Deep Dive Search ===")
    print("Strategy: Go deep into one folder path before trying others")
    print()
    
    # Stack: "Let me finish exploring this path first"
    folders_to_check = []  # This is our stack
    folders_to_check.append(("Root", file_system["Root"], ["Root"]))
    
    steps_taken = 0
    
    while folders_to_check:
        # Take the last folder from our stack (LIFO - Last In, First Out)
        current_name, current_folder, path = folders_to_check.pop()
        steps_taken += 1
        
        print(f"Step {steps_taken}: 📂 Diving into folder '{current_name}'")
        print(f"Path: {' → '.join(path)}")
        
        # Look through everything in this folder
        if "contains" in current_folder:
            # Process folders in reverse order for stack
            items = list(current_folder["contains"].items())
            
            for item_name, item_data in items:
                
                # Found a file!
                if item_data["type"] == "file":
                    print(f"   📄 Found file: {item_name}")
                    
                    # Is this the file we want?
                    if item_name == target_file:
                        final_path = path + [item_name]
                        print(f"   ✅ SUCCESS! Found '{target_file}'!")
                        print(f"   📍 Location: {' → '.join(final_path)}")
                        print(f"   📊 Total folders checked: {steps_taken}")
                        print(f"   ⚠️  Path may not be shortest (DFS explores deep first)")
                        return final_path
            
            # Add folders to stack (they'll be explored immediately)
            for item_name, item_data in reversed(items):
                if item_data["type"] == "folder":
                    new_path = path + [item_name]
                    folders_to_check.append((item_name, item_data, new_path))
                    print(f"   📂 Will dive into '{item_name}' next (added to stack)")
        
        print()  # Empty line for readability
    
    print(f"❌ File '{target_file}' not found anywhere!")
    print(f"📊 Total folders checked: {steps_taken}")
    return None

# Run the search!
result = search_file_with_dfs(file_system, "vacation.jpg")
