  • Send output through axon
  • Connected by synapses
  • Work together in networks
86 billion neurons in your brain!

🤖 Artificial Neurons

  • Receive number inputs
  • Calculate weighted sum
  • Apply activation function
  • Send number output
  • Connected in layers
Much simpler, but same idea!

The Mathematics

How Neurons Actually Work

Inside a Single Neuron

The Calculation

$$\text{output} = \sigma(\sum w_i \cdot x_i + b)$$
  • $x_i$: Inputs from previous layer
  • $w_i$: Weights (importance)
  • $b$: Bias (threshold)
  • $\sigma$: Activation function
It's like a weighted vote!

🔄 The Sigmoid Activation Function

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

What it does: Takes any number and "squishes" it to be between 0 and 1

  • Very negative input → output ≈ 0
  • Zero input → output = 0.5
  • Very positive input → output ≈ 1
💡 Why we need this: Without activation functions, our network would just be a fancy linear equation! The sigmoid adds the "nonlinearity" that lets neural networks learn complex patterns.

⚡ Making It Fast: Matrix Operations

Instead of calculating each neuron one by one...

$$\mathbf{a}^{(next)} = \sigma(\mathbf{W} \cdot \mathbf{a}^{(current)} + \mathbf{b})$$

One equation calculates ALL neurons in a layer simultaneously!

  • $\mathbf{W}$: Weight matrix
  • $\mathbf{a}$: Activation vector
  • $\mathbf{b}$: Bias vector
🚀 Speed boost: Matrix operations are 10-100x faster than loops! This is how we can train networks with millions of parameters.

🤔 Why Do Neural Networks Work?

✨ The Magic

  • Layers build up complexity gradually
  • Each layer learns useful features
  • Connections adapt during training
  • Pattern recognition emerges naturally
Universal Approximation: Deep enough networks can learn ANY pattern!

🎯 The Reality

  • Need lots of training data
  • Requires careful tuning
  • Can be fooled by edge cases
  • Not always interpretable
Powerful but not magic!

🎓 Key Takeaways

Neural networks are function approximators that learn hierarchical patterns
  • Structure: Layers of simple mathematical units (neurons)
  • Computation: Weighted sums + activation functions
  • Power: Composition creates complex decision boundaries
  • Learning: Adjust 13,002 parameters to recognize patterns
  • Applications: Image recognition, language, games, and more!
