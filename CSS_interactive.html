LESSON 3

Box Model: Border, Margin & Padding

ðŸ“š Theory

Every element is a box with four parts: content, padding, border, and margin.

Padding: Space between content and border (inside)

Border: The edge around padding

Margin: Space outside the border (creates gaps between elements)

Syntax: padding: 20px; border: 2px solid black; margin: 10px;

ðŸ’¡ Example

.box-demo {
    padding: 20px;          /* Inside spacing */
    border: 3px solid #3498db; /* Border style */
    margin: 15px;           /* Outside spacing */
    background-color: #ecf0f1;
}

ðŸ“š Theory

Pseudo Classes: Hover, Active & Focus

ðŸ“š Theory

Pseudo-classes style elements based on their state:

:hover - When mouse is over the element

:active - When element is being clicked

:focus - When element has keyboard focus

li:nth-child(odd) - Selects odd list items

ðŸ’¡ Example

    background: #3498db;
    color: white;
    padding: 10px 20px;
}

.btn:hover {
    background: #2980b9;
    transform: scale(1.05);
}

.btn:active {
    transform: scale(0.95);

LESSON 7

Tooltips: ::before & ::after

ðŸ“š Theory

::before and ::after are pseudo-elements that insert content before or after an element's content.

They require a content property to work.

Perfect for tooltips, decorative elements, and icons without extra HTML.

Use with: position: absolute for precise positioning

ðŸ’¡ Example

.tooltip {
    position: relative;
    cursor: help;
}

.tooltip::after {
    content: attr(data-tip);
    position: absolute;
    bottom: 100%;
    background: #333;
    color: white;
    padding: 5px 10px;
    opacity: 0;
    transition: opacity 0.3s;
}

.tooltip:hover::after {
    opacity: 1;
}

LESSON 8

CSS Animations

ðŸ“š Theory

CSS animations let you animate elements without JavaScript. Two steps:

1. Define keyframes with @keyframes

2. Apply animation using animation property

Properties: animation-name, animation-duration, animation-timing-function, animation-delay, animation-iteration-count

ðŸ’¡ Example

@keyframes slideIn {
    0% {
        transform: translateX(-100%);
        opacity: 0;
    }
    100% {
        transform: translateX(0);
        opacity: 1;
    }
}

.animated-box {
    animation: slideIn 1s ease-out;
}
